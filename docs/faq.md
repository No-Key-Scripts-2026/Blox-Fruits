# Blox Fruits Script FAQ

## What is Blox Fruits Script?
Blox Fruits Script is a Roblox Lua-based automation and scripting project created
for educational and experimental purposes.

## Is Blox Fruits Script free to use?
Yes, the project is open-source and intended for learning and research use.

## What programming language is used?
The project uses Lua and Luau, which are the primary scripting languages for Roblox.

## Is this project safe?
The repository focuses on clean scripting practices and does not promote misuse
or violation of Roblox terms of service.

## Can beginners use this project?
Yes, the codebase is structured to be readable and suitable for developers learning
Roblox Lua scripting and automation concepts.
